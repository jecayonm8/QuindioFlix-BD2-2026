# QuindioFlix — Proyecto Final Bases de Datos II
**Universidad del Quindío | Ingeniería de Sistemas | Semestre 2026-1**

Autor: Juan Esteban Cayon M.

## Descripción
Base de datos completa para QuindioFlix, plataforma colombiana de streaming.
Implementada en Oracle XE 21c con SQL Developer 24.3.

## Scripts incluidos
| Script | Descripción |
|--------|-------------|
| 01_CreateTables | 17 tablas con constraints, PKs, FKs y CHECK |
| 02_Tablespaces | 6 tablespaces + particionamiento REPRODUCCIONES |
| 03_DatosPrueba | 30 usuarios, 200 reproducciones, datos asimétricos |
| 04_Nucleo1 | Consultas parametrizadas, PIVOT, ROLLUP, CUBE, MVs |
| 05_Nucleo2 | Cursores, SPs, funciones, triggers |
| 06_Nucleo3 | 3 transacciones + escenario concurrencia |
| 07_Nucleo4 | 4 índices + análisis EXPLAIN PLAN |
| 08_Nucleo5 | 4 roles, 4 usuarios, PROFILE de recursos |
| 09_Vistas | 5 vistas con análisis CRUD |

## Tecnología
- Oracle Database XE 21c
- Oracle SQL Developer 24.3.1
- Oracle Data Modeler (diagrama ER)
